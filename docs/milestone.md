## Phase 1：タイムラインエディタ＋シミュレーター

**目標：ブラウザ上でパターン作成・確認ができる（転送なし）**

### タスク

- [x] プロジェクト雛形作成（Vite + Vue 3）
- [x] タイムラインエディタ
  - [x] グリッド描画（横：フレーム、縦：LED番号）
  - [x] セルクリックでカラーピッカー表示・色設定
  - [x] 範囲選択して一括塗り（ドラッグ）
  - [x] フレームのコピー・ペースト・削除・挿入
  - [x] FPS設定UI（1〜60）
  - [x] ループ設定UI（ON/OFF）
  - [x] タイムラインのスクロール
- [x] シミュレーター（Phase 1はLED直線配置固定）
  - [x] 円形ドットでLEDを表現するキャンバス（グロー効果付き）
  - [x] ループ再生
  - [x] 再生速度変更（0.25× / 0.5× / 1× / 2× / 4×）
  - [x] 一時停止・フレーム単位手動送り
  - [x] 現在フレーム番号・経過時間表示
- [x] プロジェクト保存・読み込み（`.ledproj` JSON）
- [x] LED個数設定UI

**備考：Phase 1ではレイアウトエディタは未実装。LEDは直線配置で固定。**

## Phase 2：バイナリエクスポート＋ESP32プレイヤーファーム

**目標：esptoolで書き込んで実機再生ができる**

### タスク

- [x] バイナリエクスポート機能（spec §7.2 フォーマット準拠）
  - [x] RGB→GRB変換
  - [x] ヘッダ生成
  - [x] ダウンロードボタン（「ESP32へ書き込む」）
  - [x] esptool書き込み手順をモーダルで表示
- [x] ESP32プレイヤーファームウェア（`firmware/player/`、ESP-IDF v5.x）
  - [x] バイナリ読み込み・パース（leddata パーティション 0x200000）
  - [x] WS2812B制御（espressif/led_strip コンポーネント、RMT使用）
  - [x] ループ再生
  - [x] FPS通りのタイミング制御（vTaskDelay）
